# Patient Record System using BST in C

This project implements a Patient Record Management System using
Binary Search Tree (BST) in pure C.

## Features
- Insert patient records
- Search patient by ID
- Display records (Inorder traversal)
- Update patient disease
- Delete patient record
- Count total patients

## Files
- main.c     : Main menu and user interaction
- patient.c : BST operations implementation
- patient.h : Structure definitions and function declarations

## How to Compile and Run
```bash
gcc main.c patient.c -o patient
./patient
